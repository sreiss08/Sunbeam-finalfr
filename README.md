# Sunbeam Window & Door Company - Full Website

## Overview
A comprehensive, modern website for Sunbeam Window & Door Company featuring:
- 8 fully responsive pages
- Professional dark theme with brand colors (maroon #94295D, yellow #FDF105)
- Integration of ProVia windows/doors, Hunter Douglas window treatments, and Aristocrat awnings
- Embedded booking calendar for consultations
- Full Aristocrat dealer compliance (6+ product photos, authorized dealer text and logo)
- In-house certified installer messaging throughout
- Google reviews integration
- Mobile-responsive design

## Pages Included

1. **index.html** - Home page with hero, services, reviews, stats, and CTAs
2. **about.html** - Company history, values, service area, and team information
3. **contact.html** - Embedded booking calendar, contact information, and FAQ
4. **windows.html** - ProVia window products (Endure, Aspect, Aeris collections)
5. **doors.html** - ProVia door products (entry, patio, storm doors)
6. **window-treatments.html** - Hunter Douglas shades, blinds, shutters with PowerView®
7. **awnings.html** - Full Aristocrat product line with compliance section
8. **gallery.html** - Project photo galleries organized by product category

## Installation Instructions

1. **Upload to Web Server**:
   - Upload entire sunbeam-website folder to your web hosting
   - Ensure all file paths remain intact
   - Set index.html as default homepage

2. **Update Contact Form** (if needed):
   - The booking calendar is already embedded in contact.html
   - Email address placeholder: info@sunbeamwindow.com (update as needed)

3. **Customize Content**:
   - All text can be edited directly in HTML files
   - Company colors in css/styles.css (:root section)
   - Logo and images use current sunbeamwindow.com URLs
